# 🛒 אפליקציית רשימת קניות משפחתית

*סדנה סמינריונית במדעי המחשב*

*חברי הקבוצה:* שלי קלפה, עומר לוי, עידו עזריאלי, לירון כהן, ברי בכר

*מרצה מנחה:* נדב בנו

---

## 🧾 תיאור כללי

אפליקציית Web מודרנית לניהול רשימות קניות חכמות בקבוצות משפחתיות.

כל משתמש יכול ליצור או להצטרף ל"משפחה" (קבוצת קניות), לערוך רשימות משותפות, לעדכן פריטים ולתאם בקלות עם שאר חברי הקבוצה.

---

## 🎯 תכונות עיקריות

- ✅ יצירה וניהול של מספר רשימות קניות

- 🛒 הוספה, עריכה ומחיקה של פריטים כולל שם, כמות, מחיר, תמונה ומחלקה

- 🔁 סימון פריטים כהושלמו עם אפשרות להחזירם לרשימה

- 🔍 השלמה אוטומטית חכמה באמצעות API חי מרשת סופרמרקט

- 📱 שילוב עם WhatsApp (בעזרת הספרייה whatsapp-web.js) לשליחת הרשימה בזמן אמת

- 👨‍👩‍👧 מערכת קבוצות עם תפקידים (מנהל/חבר)

- 🔐 התחברות עם חשבון Google (OAuth)

---

## 🚀 טכנולוגיות בשימוש

- *Frontend:* React

- *Backend:* Node.js

- *Database:* SQL עם phpMyAdmin

- *API חיצוני:* רמי לוי (API ציבורי)

- *שילוב בזמן אמת:* whatsapp-web.js

- *אימות משתמשים:* Google OAuth

- *כלים נוספים:* Git, Jira, Google Cloud

---

## 👨‍💻 חלוקת עבודה

- *פיתוח צד לקוח (Frontend):* שלי, עומר

- *פיתוח צד שרת (Backend):* עידו, לירון

- *אסטרטגיית חוויית משתמש וחקר מוצר:* ברי

---

## 📚 מה למדנו

- אינטגרציה עם API דינמי ואמיתי

- תכנון ותחזוקת בסיס נתונים רלציוני מורכב

- בניית ממשק משתמש רספונסיבי, נגיש וקל לשימוש

- עבודת צוות בשיטת Agile – כולל ניהול גרסאות, חלוקת משימות וביקורות קוד

- אינטגרציה עם שירותים חיצוניים וניהול תקשורת נתונים בזמן אמת
